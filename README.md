# Python Port Scanner

## Overview

Python Port Scanner is a simple cybersecurity tool developed using Python and Socket Programming. The tool scans a target host and identifies open ports within a specified range.

This project demonstrates fundamental concepts of network security, port enumeration, and TCP connection testing.

## Features

* Scan target IP address or domain
* Detect open TCP ports
* Display scan start time
* Generate scan summary
* Lightweight command-line interface

## Technologies Used

* Python 3
* Socket Programming
* Datetime Module


## Installation

```bash
git clone https://github.com/yourusername/python-port-scanner.git
cd python-port-scanner
python port_scanner.py
```

## Usage

1. Run the script.
2. Enter the target IP address or domain name.
3. View detected open ports.
4. Review the scan summary.
